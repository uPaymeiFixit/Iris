# Color Space Conversion

## Methods

### `convert.RGBtoHSV(r, g, b)`

* `r` float [0, 255] range
* `g` float [0, 255] range
* `b` float [0, 255] range

returns `[h, s, v]`

Converts RGB color space to HSV space

### `convert.HSVtoRGB(h, s, v)`

* `h` float [0, 1] range
* `s` float [0, 1] range
* `v` float [0, 1] range

returns `[r, g, b]`

Converts HSV color space to RGB space
